# User Stories

## User Story Template

**As a** [type of user]  
**I need** [some goal]  
**So that** [some reason]

---

## Example User Stories

1. As a user, I need to register an account so that I can access the platform.
2. As a user, I need to log in so that I can manage my items.
3. As a user, I need to view available items so that I can choose what I want.
4. As a user, I need to search items by category so that I can find relevant items.
5. As a user, I need to post items so that I can give away unused goods.
6. As a user, I need to comment on items so that I can communicate with others.
7. As a user, I need to update my profile so that my information stays current.
8. As a user, I need to view item details so that I can make decisions.
